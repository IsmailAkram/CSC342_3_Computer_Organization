-- READme

Open up Akram_11_14_2021_Lab_7_ALU.qar in Quartus.

Click on Hierarchy on the top right (right by Project Navigator) and click Files/

Set Akram_11_14_2021_ALU to Top-Level Entity (you can do this by right clicking on on said file).

Compile to confirm that the project runs correctly.

Close quartus and open up ModelSim.

Open Project (saved as Akram_11_14_2021_Lab_7_ALU.mpf) in ModelSim.

Add files

Run Simulation

And then put in your inputs according to the following commands derived from the "Green Pages" for the op codes:

Op codes:
0000: add
0001: addi
0010: addi unsign
0011: add unsign
0100: sub
0101: sub unsign
0110: and
0111: andi
1000: nor
1001: or
1010: ori
1011: shift left logic
1100: shift right logic
1101: shift right arith
1110: store word
1111: load word
